# 🎮 Ordeal Games — Official Website

The official website for **Ordeal Games**, an independent game studio.  
Built with React + Vite.

🔗 **Live Site:** [ordeal-games-website.vercel.app](https://ordeal-games-website.vercel.app/)

---

## About

Ordeal Games is an indie game studio behind **Highway Dash**, an endless runner arcade racing game available on Google Play. This website serves as the studio's online presence — showcasing our games, telling our story, and providing a way for players and collaborators to get in touch.

> *Dare to Dream, Dare to Game.*

---

## Pages

- **Home** — Hero section with studio branding, featured game trailer, and studio stats
- **Games** — Showcase of released titles with trailers, descriptions, tags, and store links. Includes a "Coming Soon" section for future projects
- **About** — Studio story, philosophy, and core values
- **Contact** — Contact form powered by EmailJS, plus links to all social media channels

---

## Tech Stack

- **React** — UI framework
- **React Router** — Client-side routing
- **Vite** — Build tool & dev server
- **EmailJS** — Contact form email delivery
- **Vercel** — Hosting & deployment
- **Google Fonts** — Cinzel + Outfit typography

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18+)

### Installation

```bash
git clone https://github.com/YOUR_USERNAME/ordeal-games-website.git
cd ordeal-games-website
npm install
```

### Development

```bash
npm run dev
```

Opens at `http://localhost:5173`

### Build for Production

```bash
npm run build
```

Output goes to the `dist/` folder.

## Deployment

The site auto-deploys to Vercel on every push to `main`. To set up your own:

1. Push the repo to GitHub
2. Import the project on [vercel.com](https://vercel.com)
3. Vercel auto-detects Vite — click Deploy
